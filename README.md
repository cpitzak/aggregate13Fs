# 13F Aggregator

## About
This aggregates [13F Forms](http://en.wikipedia.org/wiki/Form_13F) to gain insights about what the guru money managers and investors
are buying and selling each quarter.


## Run

   	$ java -jar build/13FAggregator.jar
